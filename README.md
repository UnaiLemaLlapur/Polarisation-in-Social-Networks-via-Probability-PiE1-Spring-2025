# Polarización en Redes Sociales via Probabilidad por Unai Lema y Sebastián Luna

Este proyecto explora cómo los algoritmos de recomendación en redes sociales pueden influir en la polarización política, utilizando modelos probabilísticos para analizar patrones de interacción y consumo de contenido.

## Objetivos

- **Entender** cómo los algoritmos de recomendación modifican las opiniones políticas de los usuarios.
- **Modelar** la interacción entre usuarios y contenido utilizando herramientas probabilísticas.
- **Evaluar** el impacto de diferentes estrategias de recomendación en la polarización de opiniones.

## Herramientas Utilizadas

- Lenguajes de programación: Python
- Librerías: NumPy, Yogi
- Visualización: Matplotlib
